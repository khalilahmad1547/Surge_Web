Install Pre-requests

Pre-requests for Surge is Node.js. Node.js is a JavaScripe library which have a pakage manager named as npm. npm made easy for us to install different pakages and to manage them.

Go a head, downloade and install Node.js from here

Install Surge

To install surge we can follow below steps.

  • Open your cmd
  • Now run the command "npm install -g surge"

Creating Surge Account and Hosting Web

Go inside the folder in which there is content for the web and run command 'surge' Surge will asks for email and password and than it will uploade the website to specified domain (by you).
